# Twenty4 Fitness 🏋️‍♂️

Landing Page responsiva desarrollada para el Laboratorio #1 del curso **Programación en Ambiente Web I**.

## 📋 Información General

* **Proyecto:** Twenty4 Fitness
* **Tipo:** Landing Page responsiva
* **Curso:** Programación en Ambiente Web I
* **Universidad:** Universidad Técnica Nacional
* **Estudiante:** Luis Giovanni Sandi Azofeifa
* **Año:** 2026

## 🚀 Características

* Diseño completamente responsivo.
* Navegación accesible mediante teclado.
* Modo oscuro y modo claro.
* Persistencia del tema mediante Local Storage.
* Panel de accesibilidad.
* Carrusel interactivo de imágenes.
* Navegación móvil mediante menú hamburguesa.
* Botón para volver al inicio de la página.
* Integración con Google Maps.

## ♿ Características de Accesibilidad

* Uso de etiquetas HTML5 semánticas.
* Navegación mediante teclado (Tab y Enter).
* Textos alternativos (`alt`) en imágenes.
* Uso de atributos ARIA.
* Ajuste del tamaño del texto.
* Modo de alto contraste.
* Lectura de contenido en voz alta.

## 🛠️ Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript (ES6)
* Flexbox
* CSS Grid
* Local Storage
